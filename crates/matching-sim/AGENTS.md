# matching-sim

Simulation harness for the matching system.

## Purpose

Runs simulations to study market dynamics, agent behavior, and solver performance
over multiple time steps.

## Status

Work in progress. Intended for:
- Multi-epoch market simulations
- Agent-based modeling
- Performance analysis over time
- Market maker strategy evaluation

## Dependencies

- `matching-engine`: Core types
- `matching-solver`: Solver implementation
- `matching-scenarios`: Scenario generation
